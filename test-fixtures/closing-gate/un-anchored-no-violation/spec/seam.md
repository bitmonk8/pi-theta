# Seam (fixture spec page — SEAM)

This page owns a cross-leaf seam contract. The adapter MUST surface the seam to
its consumer without anchoring the obligation to a numbered REQ-ID or a
diagnostics-registry code.

So this is an un-anchored normative MUST, and it is enumerated in the Code-keyed
obligation areas table with a real closing leaf — so the gate stays green.
