## Problem

`H4b` defines the shared response-programming surface as a single leaf carrying seven heterogeneous injection categories `(a)`вЂ“`(g)` under one `Ships when` gate. Categories `(a)`вЂ“`(e)` are the harness's foundational scripting surface consumed by nine downstream leaves, whereas `(f)`/`(g)` model the completed-invoke-child and subagent-mode scripting contracts that forward-reference the not-yet-authored `V4f`/`V9i` slices. Because the single gate requires the determinism check plus functional-effect assertions for all of `(a)`вЂ“`(g)` simultaneously, the broadly-consumed `(a)`вЂ“`(e)` contract has no observable completion state independent of the heavier, downstream-coupled `(f)`/`(g)` modelling.

## Issue introduction

**Verdict:** present-since-inception
**Introducing commits:** `fd6169b` (2026-06-12, "pi-loom plan: resolve \"H4a bundles three independent units into one leaf\"")
**History:** The `H4b` leaf file was created in `fd6169b` вЂ” itself a split-off from `H4a` вЂ” already carrying all seven injection categories `(a)`вЂ“`(g)` under a single `Ships when` gate, including the `(f)`/`(g)` categories that forward-reference the not-yet-authored invoke-child / subagent-mode slice contracts (then labelled `V4c`, since relabelled to `V4f`/`V9i`). The two later commits touching this file (`eaa5000` on 2026-06-16, `098ec80` on 2026-06-17) only relabelled the consuming-vector IDs (`V13c`в†’`V14b` for the parallel batch, `V4c`в†’`V4f` for ERR-13); they did not introduce the bundling. The defect is therefore present since the leaf's inception, not introduced by a later edit.

## Solution approach

Split `H4b` so the broadly-consumed `(a)`вЂ“`(e)` scripting contract reaches an observable `Ships when` completion independent of the forward-referenced `(f)`/`(g)` modelling. Narrow `H4b` to categories `(a)`вЂ“`(e)`, leaving its determinism and functional-effect gate covering only those categories, then add a new H-phase leaf (depending on `H4b`) that models the `(f)`/`(g)` invoke-child and subagent-mode scripting contracts on the now-stable base. Partition the gate assertions so neither leaf asserts the other's categories, and add the new leaf to the `Deps` of its `(f)`/`(g)` consumers (`V4f`, `V9i`).
