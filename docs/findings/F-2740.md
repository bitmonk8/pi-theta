## Finding

`V17a`'s downward-only-propagation test bullet asserts that "cancellation propagates parent â†’ child invoke / in-flight query / in-flight tool call, and never child â†’ parent." ## Consequence

**Severity:** advisory

An implementer reading `V17a`'s propagation bullet in isolation sees it name `invoke`/query/tool-call kinds that do not yet exist, with no statement that they are witnessed via the `V8a` seam and `H4a`/`H4b` doubles.
