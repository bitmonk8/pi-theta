## Finding

The `ERR-8` Tests bullet in both `V9i` (`docs/plan_topics/V9i-subagent-isolation.md`) and its paired tests leaf `V9i-T` (`docs/plan_topics/V9i-T-subagent-isolation.md`) closes with the standalone sentence "This is the live-surface confirmation `V4c` defers from the `H4a` double." The same fact is already carried by the bullet's lead-in parenthetical, "`ERR-8` (delegated live-carrier witness for `V4c`'s ERR-8/ERR-12 deferral)", and the body already states the behaviour is real-host-only and witnessed at the manual smoke gate (`real-host-smoke-gate.md` criterion (c)).

The trailing sentence therefore adds no acceptance content. Because it sits inside a normative Tests bullet, it reads as a deliverable rather than the cross-leaf context it actually is, and it duplicates verbatim across both paired files. The bullet's witness obligation is complete without it.

## Issue introduction

**Verdict:** introduced-by-commit (refining a phrasing present since the bullet's inception)
**Introducing commits:** `c768005` (`pi-loom spec-plan: fix F-1363 — relocate V9i ERR-8 real-session witness to smoke gate`, 2026-06-25)
**History:** The ERR-8 bullet first appeared on `V9i` in `7bd65a7` (2026-06-18) as a single sentence whose em-dash tail already carried the "the live-surface confirmation `V4c` defers from the `H4a` double" clause, partly redundant with the same bullet's lead-in parenthetical. `f892019` (2026-06-23, F-1243) added the paired ERR-8 bullet to `V9i-T`. `c768005` (2026-06-25, F-1363) rewrote the bullet in both files into its current two-sentence form, splitting that clause off into the standalone trailing sentence this finding flags and adding the real-host-only / smoke-gate framing. The standalone trailing sentence in its present, separable form entered both paired files at `c768005`.

## Solution Space

**Shape:** single

### Recommendation

Strike the trailing sentence "This is the live-surface confirmation `V4c` defers from the `H4a` double." from the `ERR-8` Tests bullet in both `docs/plan_topics/V9i-subagent-isolation.md` and `docs/plan_topics/V9i-T-subagent-isolation.md`. The bullet's lead-in parenthetical ("delegated live-carrier witness for `V4c`'s ERR-8/ERR-12 deferral") and its existing smoke-gate clause retain the V4c/H4a deferral relationship; no replacement text is needed. The edit is confined to the two named plan files; the spec is read-only for this fix.

## Consequence

**Severity:** cosmetic

An implementer reading the ERR-8 Tests bullet meets the same cross-leaf deferral fact twice; the second statement, placed in a normative Tests field, can be misread as an additional acceptance obligation. No implementer would diverge on the actual behaviour — the duplication is pure wording cruft.
