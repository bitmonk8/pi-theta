## Finding

- **Description:** The shape mandates `YYYY-MM-DD` but never says whose clock or which zone; two contributors in different zones date the same commit differently.
- **Suggested fix:** State the time-zone/clock basis (e.g. UTC, or the commit's authored date), or note it is non-normative.
