## Problem

The Tests bullet `The six failure templates render verbatim …` in both `V11f-T-binder-retry-taxonomy.md` and the paired `V11f-binder-retry-taxonomy.md` leads with no citation token — no `PREFIX-N` REQ-ID, no `HC3-<x>` inline label, no `loom/…` diagnostic code, and no `cka-N` code-keyed token. It closes the `determinism-cancellation-failure.md` §Failure-mode templates obligation (renderers emit each non-cancelled failure-row template verbatim), but the `HC3-a … HC3-e` labels the leaf cites cover only the per-class retry budget and taxonomy, not the template-rendering MUST.
