## Problem

`H1a` provisions an `npm run build` script and a `pi.extensions = ["./extensions"]` manifest entry, and both `H1a` and `H4a` pin `extensions/index.ts` as the file Pi resolves under that discovery directory — but the plan never states how the `npm run build` output relates to the module Pi actually executes. A fresh implementer reading `H1a`, which lists `npm run build`, the `["./extensions"]` directory, and `"type": "module"` side by side, can reasonably conclude that `npm run build` compiles `extensions/index.ts` to a runtime artifact and that discovery should resolve that build output, shipping a manifest/spec that name a `.ts` entry while the executed file is a compiled `.js`. The mismatch passes every automated gate — the `H4a` harness imports the factory in TypeScript and never drives the real Pi loader or the build target — and surfaces only at the manual real-host smoke gate.

Each edit was internally coherent for its own purpose — `c6a664e` added the build/test/typecheck scripts; `e7c1b1e` established the loaded-`.ts`-entry discovery contract — but `e7c1b1e` pinned the loaded-entry contract without reconciling it against the pre-existing `npm run build` script, so the gap is the unstated interaction between the two commits rather than a defect introduced wholesale by either one.

## Solution approach

In `H1a`'s Adds — the enumeration that already lists `npm run build`, `pi.extensions = ["./extensions"]`, and `"type": "module"` — state the loaded-entry relationship: Pi resolves the authored on-disk `extensions/index.ts` directly under the `["./extensions"]` discovery directory, so `npm run build`'s output is not the module Pi executes at load time. Mirror the same fact in `H4a`'s Adds where it pins `extensions/index.ts` as the file Pi resolves, so the entry-shim and manifest leaves agree on what the host loads versus what `npm run build` produces. Source the loaded-`.ts`-entry fact from `extension-bootstrap-and-per-loom.md` §"Extension entry point", which already names `extensions/index.ts` as the discovered entry.
