## Finding

`slice(0,256)` on `String(error)` can leave a lone surrogate as the final code unit, which falls in none of the escape ranges and passes through; the spec never says whether truncation may split a surrogate or must back off to a whole code point, and the acceptance fixture tests only lengths 256/257.

**Suggested fix:** State whether truncation may split a surrogate (pass the lone surrogate through) or must round down to the last whole code point, with a matching fixture.
