"error for explicit references, warning never" is non-standard; readers may misread it as prohibiting warnings rather than the conventional-location case emitting no diagnostic.

**Suggested fix:** Rephrase: `E` for explicit references; conventional locations emit no diagnostic when missing (the code does not fire for those sources).
