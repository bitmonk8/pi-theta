## Finding

`V7c`'s **Ships when** field reads: "`npm test` asserts the byte-identical vector for each of the eight categories." This claims a full-string byte-identical equality assertion for all eight placeholder-rendering categories. Category 8 (`loom/load/host-incompatible`, `node-floor`) does not use full-string equality: its rendering carries an implementation-defined `<observed>` segment, so the leaf's own **Tests** bullet asserts it via the §8 anchored partial-match pattern `^host incompatible \(node-floor\): observed (.*), required >=22\.19\.0$` — "never a full-*Message*-string equality (non-conformant for category-8 rows)."

The gate criterion (Ships when) therefore contradicts the test it gates: the Tests bullet correctly specifies a prefix/suffix-byte-identical match with an anchored partial-match on the variable segment for category 8, while Ships-when blankets all eight categories under "byte-identical vector." The Tests bullet sits directly above and disambiguates, so a careful reader reconciles, but the gate field as written overstates the mechanism for category 8.

## Consequence

**Severity:** advisory

The Ships-when gate criterion contradicts the leaf's own category-8 Tests bullet. An implementer reading Ships-when in isolation could assert full byte-identical equality for category 8 and fail against the implementation-defined `<observed>` segment; the adjacent Tests bullet provides the correct partial-match observable, so the divergence is bounded and recoverable rather than build-breaking.

## Solution Space

**Shape:** single

### Recommendation

In `docs/plan_topics/V7c-placeholder-rendering.md`, rewrite the **Ships when** field so the gate criterion matches the category-8 mechanism the Tests bullet already specifies. Replace:

> **Ships when.** `npm test` asserts the byte-identical vector for each of the eight categories.

with text that distinguishes categories 1–7 (full byte-identical) from category 8 (byte-identical prefix/suffix with an anchored partial-match on the `<observed>` segment), e.g.:

> **Ships when.** `npm test` asserts the byte-identical vector for categories 1–7, and for category 8 asserts the byte-identical prefix/suffix with the §8 anchored partial-match on the implementation-defined `<observed>` segment.

The replacement is confined to the Ships-when field; the Tests bullets and Adds field already carry the correct category-8 detail and need no change.
