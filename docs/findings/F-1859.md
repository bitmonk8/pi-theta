## Finding

The budget value is "implementation-pinned (e.g. 60 s)" with no floor/ceiling, so 1 ms or 24 h both conform and no test can distinguish a correctly-calibrated budget.

**Suggested fix:** Pin a normative range (e.g. ≥30 s, ≤300 s) or explicitly demote the value to non-normative guidance, keeping only the expiry behaviour normative.
