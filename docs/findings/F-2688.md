## Finding

The README status table is described as "replacing the current prose `## Status` section on first write," presupposing README exists with a `## Status` section, while the same rule also says artifacts are "created on first use if absent" — contradicting the presupposition and leaving the README-exists-but-no-`## Status` case undefined. Suggested direction: make the instruction state-independent — create the status table when absent; replace only an existing `## Status` section.
