## Finding

The `V9q-T` Tests and the mirrored `V9q` Tests both assert that, on the watcher
stopped-delivering terminal case, "the `LoomRegistry` stays live and dispatchable
through arm (a) of `readDrainState` against the last-published snapshot, and **no**
`LoomRegistry` drain-state tag is written." `readDrainState`, the drain-state tag,
and the three-arm drain-state contract are production symbols owned by `V9m`
(`LoomRegistry` drain-state contract); the `LoomRegistry` itself is built and
swapped by `V9b`. `V9m` declares `Deps. V9m-T, V9b`, so `V9m` transitively pulls
`V9b`.

`V9q` declares `Deps. V9q-T, V8e, V7b, V7d` and `V9q-T` declares `Deps. V8e, V7b,
V7d`. Neither reaches `V9m` or `V9b`. Under the pickability predicate
(`conventions.md` §Leaf format **Deps**), `V9q-T` becomes pickable as soon as
`V8e`/`V7b`/`V7d` are complete — before `V9m`/`V9b` exist — so its suite references
production symbols (`readDrainState`, the drain-state tag, the dispatchable
`LoomRegistry`) that are not yet declared.

The sibling drain-state-touching leaves already carry the edge: `V9h`/`V9h-T` and
`V9p`/`V9p-T` each list `V9m` in `Deps` precisely because they reference drain
state (`V9p-T` even notes "drain state is owned by `V9m`'s `LoomRegistry`
contract"). `V9q`/`V9q-T` are the outliers that assert against the same contract
without declaring the dependency.

## Consequence

**Severity:** blocking

`V9q-T` can be picked and its `<id>-T-complete` tag claimed before `V9m`/`V9b`
exist, but its suite references not-yet-declared production symbols
(`readDrainState`, the drain-state tag, the dispatchable `LoomRegistry`), so it is
a compile-time red — which `conventions.md` §"Fail red for the intended reason"
explicitly disqualifies from satisfying the tests-task gate. The leaf cannot be
legitimately completed in its declared dependency position.

## Solution Space

**Shape:** single

### Recommendation

Add `V9m` to the **Deps.** field of both leaves; `V9m` transitively carries `V9b`
(its own `Deps` list `V9m-T, V9b`), so the dispatchable-`LoomRegistry` and
`readDrainState`/drain-state-tag symbols all become declared before the suites
compile.

- In `docs/plan_topics/V9q-T-watcher-terminated-recovery.md`, the **Deps.** value
  `` `V8e`, `V7b`, `V7d` `` gains `` `V9m` `` (e.g. `` `V9m`, `V8e`, `V7b`, `V7d` ``).
- In `docs/plan_topics/V9q-watcher-terminated-recovery.md`, the **Deps.** value
  `` `V9q-T`, `V8e`, `V7b`, `V7d` `` gains `` `V9m` `` (e.g. `` `V9q-T`, `V9m`, `V8e`, `V7b`, `V7d` ``).

This mirrors the edge `V9h`/`V9h-T` and `V9p`/`V9p-T` already carry for the same
`V9m`-owned drain-state symbols.
