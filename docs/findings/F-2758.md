## Finding

The **Ships when** field of `docs/plan_topics/V4f-no-rollback.md` is a gate-criterion field: it states the observable condition under which the leaf may be tagged complete. Its first sentence does exactly that — `npm test` proves the enumerated `ERR-13` cases hold, a completed callee stays final after a downstream terminal event, with no side effect unwound and no compensating turn injected.

The field then appends a second sentence — "The no-rollback guarantee is architectural: it rests on the runtime containing no compensating/rollback path, and the tests witness it on the enumerated cases rather than exhaustively." — which is methodology/rationale, not a gate criterion. The same rationale is already stated in **Adds** ("The no-rollback guarantee is architectural: the runtime contains no compensating path, so a completed callee's side effect survives a downstream terminal event by construction…") and in **Tests** ("The guarantee rests on the runtime having no compensating/rollback path, so the tests witness it on these enumerated authoring sites rather than proving the absolute absence of a compensating path exhaustively."). The trailing sentence in Ships-when is therefore redundant and misplaced; it adds no gate information and dilutes the field's single purpose.

## Consequence

**Severity:** cosmetic

The duplicated rationale is pure organisation noise in a gate field; no implementer is misdirected by it (the gate condition is unambiguous in the first sentence). The cost is only that the Ships-when field reads as carrying more gate substance than it does, and the rationale now lives in three fields where one or two suffice.

## Solution Space

**Shape:** single

### Recommendation

In `docs/plan_topics/V4f-no-rollback.md`, strike the trailing sentence of the **Ships when** field — "The no-rollback guarantee is architectural: it rests on the runtime containing no compensating/rollback path, and the tests witness it on the enumerated cases rather than exhaustively." — leaving the field as its first sentence only ("`npm test` proves the enumerated `ERR-13` cases hold — a completed callee stays final after a downstream terminal event across the sampled authoring sites, with no side effect unwound and no compensating turn injected."). The struck rationale survives verbatim in **Adds** and in the closing sentence of the **Tests** `ERR-13` bullet, so no information is lost.
