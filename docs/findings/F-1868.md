## Finding

Panics route via group B; the `code?` field's `loom/runtime/*`/`loom/load/*` examples don't obviously belong to any group-A event, and which group-A kinds populate `code` (and with what value) is unstated.

**Suggested fix:** State which group-A kinds populate `code` and with what value (or that group-A omits it), removing the misleading load/runtime-code examples.
