# Code registry parse

## Code registry

The V1 diagnostic registry is split across four pages — the `theta/parse/*` table on this page and the [`theta/load/*`](./code-registry-load.md), [`theta/runtime/*`](./code-registry-runtime.md), and [`theta/host/*`](./code-registry-host.md) continuation tables — which together enumerate every diagnostic the V1 spec defines. The column legend, the closed *Phase* taxonomy, and the *E/W* severity carve-out that govern all four tables are defined in [Diagnostic shape — Code registry rules (normative)](./diagnostic-shape.md#column-legend).
