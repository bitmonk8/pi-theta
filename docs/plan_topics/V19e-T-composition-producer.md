# `V19e-T` — Loom composition producer: parsed .loom → runnable slash command (tests)

**Spec.** [`../spec_topics/pi-integration-contract/extension-bootstrap-and-per-loom.md`](../spec_topics/pi-integration-contract/extension-bootstrap-and-per-loom.md), [`../spec_topics/pi-integration-contract/registration-steps.md`](../spec_topics/pi-integration-contract/registration-steps.md), [`../spec_topics/pi-integration-contract/conversation-drive.md`](../spec_topics/pi-integration-contract/conversation-drive.md), [`../spec_topics/slash-invocation.md`](../spec_topics/slash-invocation.md) (with their transitive normative closure).

**Adds.** The failing test suite pinning the per-loom runnable producer — a parsed `.loom` ([`V19a`](./V19a-whole-program-parser.md) frontmatter + body AST) mapped to a [`H4a`](./H4a-factory-shell-and-harness.md) `LoomFixture { slashName, run }` whose `run` binds the [`V11a`](./V11a-binder-model-resolution.md) frontmatter binder (when applicable) and then drives [`V19d`](./V19d-effectful-statement-wiring.md)'s effectful executor against the appropriate conversation (prompt-mode via [`V12a`](./V12a-slash-dispatch.md)/[`V9c`](./V9c-conversation-drive.md), subagent-mode via [`V9i`](./V9i-subagent-isolation.md)'s `AgentSession` spawn) — plus the minimum production stub (the widened [`reload-wiring.ts`](../../src/extension/reload-wiring.ts) `ParsedLoom` and producer signature) needed for the suite to compile and red on its own primary assertions.

**Tests.**
- `Convention:` ([`conventions.md`](./conventions.md) §"REQ-ID discipline" — the [`governance.md` GOV-22](../spec_topics/governance/req-id-prefix-table-active-b.md#gov-22) un-anchored-obligation residue routed to [release-time residue-inspection](./conventions.md#release-time-residue-inspection) item 5; source obligation: [`extension-bootstrap-and-per-loom.md` §"Per-loom registration"](../spec_topics/pi-integration-contract/extension-bootstrap-and-per-loom.md) — the per-loom runnable-producer obligation "the slash-command `handler` runs the binder (when applicable) and then the loom interpreter against the appropriate conversation"): a parsed `.loom` (`V19a` frontmatter + body AST) maps to a registered `LoomFixture` with the correct `slashName` whose `run` runs the binder then the loom interpreter against the appropriate conversation. This obligation is phrased declaratively (no literal `MUST`/`MUST NOT` token) and carries no `PREFIX-N` REQ-ID or `loom/...` registry code, so it takes **no** coverage-matrix `cka-<n>` row — **this leaf closes no new coverage-matrix row**; the obligation is witnessed by this executable test and by release-time residue-inspection item 5.
- Prompt-mode drive ([`slash-invocation.md` SLSH-2](../spec_topics/slash-invocation.md#slsh-2), [`conversation-drive.md` PIC-53](../spec_topics/pi-integration-contract/conversation-drive.md#pic-53), integration witnesses — not re-closing SLSH-2/PIC-53, owned by [`V12a`](./V12a-slash-dispatch.md)/[`V9c`](./V9c-conversation-drive.md)): the `run` of a prompt-mode parsed `.loom` drives exactly one real prompt turn via `V19d`'s executor against the user conversation and extracts the trailing-turn `Ok(string)`, leaving one appended turn.
- Subagent-mode drive ([`subagent.md`](../spec_topics/pi-integration-contract/subagent.md), [`V9i`](./V9i-subagent-isolation.md)'s subagent-mode spawn-and-drive seam `PIC-40`/`PIC-42`/`PIC-43`, integration witness — not re-closing them): the `run` of a subagent-mode parsed `.loom` spawns an isolated `AgentSession` through `V9i`'s spawn seam and drives the executor against that private session rather than the user conversation.
- Binder-before-interpreter ordering ([`V11a`](./V11a-binder-model-resolution.md)'s binder seam, integration witness): `run` binds the frontmatter binder before entering the loom interpreter, asserted by observing the bind step commit ahead of the executor's first statement.
- `ParsedLoom` widening (`Convention:` — Class-2 cross-leaf seam consumed by [`H8a`](./H8a-live-production-acceptance.md)): [`reload-wiring.ts`](../../src/extension/reload-wiring.ts) `ParsedLoom` is widened from `{ slashName }` to carry the `V19a` frontmatter + body AST + the producer `run`, the seam `H8a`'s `session_start` registration consumes.

**Deps.** `V19d`, `V19a`, `V6a`, `V9b`, `V11a`, `V9c`, `V9i`, `V12a`, `V12b`, `V9n`, `M` — the effectful executor and whole-program parser the producer composes, the frontmatter contract, the registration/reload wiring whose `ParsedLoom` it widens, the binder-model resolution, the prompt-mode and subagent-mode drivers, slash dispatch and its top-level `Err` chain, the prompt-mode transport-error mapping, and the minimal pipeline whose regex producer it supersedes.
